# Personal Portfolio Website

Developed by **Subham Khandual**.

A modern, interactive portfolio website built with React.js, Vite, and Framer Motion. 

### Key Features:
- **3D Certificate Carousel**: Showcasing NPTEL certifications with interactive 3D rotation.
- **Projects Showcase**: Integrated with AI chatbots (Susri Healthcare & Suraksha).
- **Backend Integration**: Connected to a local MongoDB instance for handling "Let's Connect" messages.
- **Modern UI**: Dark-themed glassmorphism design with smooth animations.

### Tech Stack:
- **Frontend**: React.js, Vite, Framer Motion, Lucide-React, CSS Modules.
- **Backend**: Node.js, Express.
- **Database**: MongoDB.
